# How to use

- General:

```bash
dasgoclient --query="dataset=/VH*/*Summer20UL*/MINIAODSIM"
```

- use `instance=prod/phys03`, if we are searching for user published dataset

```bash
dasgoclient --query="dataset=/VH*/*Summer20UL*/USER instance=prod/phys03"
```

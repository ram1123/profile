# Brief info about singularity


# Instructions

One just need to type `cmssw-cc6` after login to the lxplus to get the sl6 image. Then start working as usual.

```bash
ssh lxplus.cern.ch
cmssw-cc6
cmsrel CMSSW_xx_x_xx
```
